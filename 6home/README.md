# 砸六家

六人实时联机纸牌游戏，包含房间码、固定组队、混儿判定、倒计时托管、掉线重连、快捷聊天、对局记录及 Docker 部署。

## 本地启动

```powershell
docker compose up --build
```

打开 `http://localhost:4188`。同一台电脑可使用不同浏览器或无痕窗口模拟多名玩家。

## NAS 部署

1. 将整个项目复制到 NAS。
2. 确保 NAS 已安装 Docker Compose。
3. 在项目目录运行 `docker compose up -d --build`。
4. 访问 `http://NAS地址:4188`。

数据保存在 `./data`，升级前备份该目录即可。外网使用时，请通过 NAS 反向代理配置 HTTPS。

## 单人测试机器人

房主单独创建房间后，可在等候房间点击“补齐5个测试机器人”。机器人会自动准备；你准备后即可开始游戏。它们仅用于测试：领出时出最小单牌，跟牌时自动过。

## 规则约定

- 首轮与新一轮可自由出 1～9 张同点数组合；跟牌必须数量一致且更大。
- 大王、小王、3、2 是混儿；3只替代2，2只替代A，大小王可按规则向下替代，并且组合中必须存在真实目标牌。
- 大王加小王是特殊“对王”，为最大的两张组合。
- 领出者不能过；超时自动出最小单牌。跟牌超时自动过。
- 首位出完者为“大供”。包含大供的一队率先全部出完则获胜；另一队率先全部出完则平局。
