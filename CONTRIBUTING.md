# Contributing

Thanks for helping improve Readme Doctor.

## Local setup

This project is a static browser app. Open `index.html` directly, or serve the folder with any simple static server.

## Checks

Run the test script before opening a pull request:

```bash
npm test
```

## Pull requests

- Keep the app dependency-free unless a new dependency clearly improves the user experience.
- Keep suggestions plain-language and useful for small GitHub projects.
- Test the main flow: paste README, review checks, copy report, copy or download rewrite.
